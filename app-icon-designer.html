📋 How to Create Your App Icon

  1. Choose a design above or create your own inspired by these
  2. Create a 1024x1024px PNG image (no transparency)
  3. Save it as icon-1024.png in the ao-vault folder
  4. Run: chmod +x generate-app-icons.sh && ./generate-app-icons.sh
  5. All required sizes will be generated automatically!

🎨 Design Tips

  1. Keep it simple - Icons are small, avoid tiny details
  2. Use bold colors - Must be visible on any background
  3. No rounded corners - Apple adds them automatically
  4. Test at small sizes - Should be recognizable at 60x60px
  5. Consider dark mode - Icon should work on dark and light backgrounds

💡 Brand Elements

  • 🔒 Lock/Vault symbol - represents privacy and security
  • 📚 Book/Library symbol - represents fanfiction collection
  • AO letters - connection to Archive of Our Own
  • Purple/Gold colors - fanfiction community colors
  • Heart symbol - for the "Give Thanks" feature
