📱 How to Use This Icon

  1. Take a screenshot of the large icon above
  2. Open the screenshot in Preview or any image editor
  3. Crop to just the icon (including the rounded corners)
  4. Resize to exactly 1024×1024 pixels
  5. Save as icon-1024.png in the ao-vault folder
  6. Run: chmod +x generate-app-icons.sh && ./generate-app-icons.sh
  7. All App Store sizes will be generated automatically!
